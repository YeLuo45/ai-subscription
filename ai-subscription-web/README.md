# AI Subscription Web

AI 订阅内容聚合 Web 端。

## 功能特性

- 订阅源管理（预设 RSS + 自定义 RSS/API）
- 定时内容抓取
- AI 摘要生成（多模型适配器）
- GitHub Trending 定时爬取
- 本地 IndexedDB 存储

## 技术栈

- React 18 + Vite + TypeScript
- Ant Design 5
- dayjs

## 本地运行

```bash
cd ai-subscription-web
npm install
npm run dev
```

## 构建

```bash
npm run build
```
